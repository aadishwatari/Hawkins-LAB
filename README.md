<h1>Hawkins LAB</h1>
As a huge stranger things fan (only watched till season 4 lmao), Hawkins always felt like a fever dream. I always wanted to be part of a mysterious suburban-industrial town where I could do life changing "side-quest". This website is my imagination of Hawkins LAB... in this, the main character explores the map while being chased by a monster (Demogorgan). Also throughout the lab, there are different portals that takes the main character to another world or environment.
